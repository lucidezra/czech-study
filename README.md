# 🇨🇿 Czech Citizenship Exam Study Guide

Interactive flashcard and quiz app covering all 30 topics of the Czech citizenship civics exam (*zkouška z českých reálií*).

## Features

- **150+ flashcards** across all 30 exam topics
- **Quiz mode** with randomised multiple-choice questions
- **Filter by area or topic** to focus on weak spots
- **Hints** and instant feedback on every question
- Progress tracking with known/unknown counts

## Topics Covered

- **Občanský základ** (16 topics) — customs, transport, health, education, politics, law, employment, tax, social security
- **Geografické informace** (7 topics) — geography, nature, regions
- **Historické a kulturní informace** (7 topics) — history, culture, famous figures

## Local Development

```bash
npm install
npm run dev
```
